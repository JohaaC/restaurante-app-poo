# Sistema de Gestión de Restaurante

## Estudiante
Johana Lorena Castro Alban 

## Descripción
Este proyecto fue desarrollado en Python utilizando Programación Orientada a Objetos (POO). El sistema permite registrar productos y clientes de un restaurante y mostrar la información almacenada.

## Estructura del proyecto

restaurante_app/
│
├── modelos/
│ ├── producto.py
│ └── cliente.py
│
├── servicios/
│ └── restaurante.py
│
└── main.py

### Modelos
- Producto: representa los productos disponibles en el restaurante.
- Cliente: representa los clientes registrados.

### Servicios
- Restaurante: administra productos y clientes.

### Archivo principal
- main.py: crea los objetos y ejecuta el programa.

## Reflexión
La modularización permite organizar mejor el código, facilitando su mantenimiento y comprensión. La separación de responsabilidades ayuda a que cada clase tenga una función específica, haciendo que el software sea más ordenado, reutilizable y escalable.
